O que é HTML?

O HTML, diferentemente do que muitos iniciantes podem pensar, não é uma linguagem de programação e sim uma linguagem de marcação de hipertexto. Sua principal função é fornecer uma estrutura para o website, ou seja, demarcar cada elemento da página sem se preocupar necessariamente com a estética (parte que será desenvolvida através do CSS).

Estrutura básica do HTML

A estrutura básica de um arquivo HTML deve conter, no mínimo, as seguintes tags: 

        
            <html>
                <head>
                    <title>
                    </title>
                </head>
                <body>
                </body>
            </html>

  1. A tag <html> indica que tudo o que estiver dentro dela são elementos que fazem parte da linguagem HTML.
  2. Na tag <head> estão as informações do site e são ocultados para o usuário.
  3. A <title> representa o título da página que fica na aba do navegador.
  4. Por fim, a tag <body> engloba todo o corpo da página, ou seja, todo o conteúdo visível pelo leitor.

Tags

As tags são essenciais para a construção de um arquivo HTML, pois são elas que informam para o navegador como o texto dentro delas deve ser interpretado. Elas são formadas por uma palavra entre os sinais de "menor que" (<) e "maior que" (>) e, para fechar a tag, é colocada uma barra (/) antes dessa mesma palavra. Exemplo: <h1></h1>.

Nesse exemplo, a tag <h1> transforma o texto em um título de tamanho h1 (maior tamanho) e é usado para determinar o assunto principal da página. Para visualizar como fica o resultado, role para cima ou clique aqui e veja o título "Introdução ao HTML".

Textos

Para formatar textos, o HTML possui diversas tags. Aqui estão algumas delas e seus respectivos exemplos:

  1. <p>

    Tag para criar parágrafos.

  2. <h1> até <h6>

    Tags usadas para definir títulos e subtítulos:

    h1

    h2

    h3

    h4

    h5
    h6

  3. <strong>

    Tag para aplicar negrito.

  4. <u>

    Tag para aplicar sublinhado.

  5. <i>

    Tag para aplicar itálico.

  6. <mark>

    Tag para aplicar marca-texto.

  7. <sup>, <sub>

    Tags para aplicar sobrescrito e subscrito.

  8. <blockquote>

    Tag para criar citações.

Imagens

Para adicionar imagens à página, utiliza-se a tag <img> com o atributo src="", onde será colocado o nome do arquivo ou o seu link, caso pegue da internet.

Para inserir imagens salvas no próprio dispositivo, basta garantir que elas estejam no mesmo diretório (pasta) que o arquivo HTML e escrever o nome junto com sua extensão:

<img src="html-imagem.png">

Também é possível usar imagens direto da internet, apenas colocando sua url:

<img src="https://upload.wikimedia.org/wikipedia/commons/thumb/6/61/HTML5_logo_and_wordmark.svg/800px-HTML5_logo_and_wordmark.svg.png">

Listas ordenadas e não ordenadas

Listas ordenadas

Nas listas ordenadas, como o próprio nome já diz, os itens são colocados em ordem. São utilizadas as tags <ol>, para criar a lista, e <li> para especificar cada item. Exemplo: 

    
        <ol>
            <li>
                Primeiro item
            </li>
            <li>
                Segundo item
            </li>
            <li>
                Terceiro item
            </li>
        </ol>
  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Listas não ordenadas

As listas não ordenadas são organizadas em tópicos e seguem a mesma lógica de construção das listas ordenadas, porém utilizando a tag <ul> ao invés de <ol>. Exemplo: 

        
            <ul>
                <li>
                    Item
                </li>
                <li>
                    Item
                </li>
                <li>
                    Item
                </li>
            </ul>

Links

Os links nos permitem navegar pela internet e, para criá-los, é usada a tag <a>. No entanto, para especificar a página a qual o usuário será direcionado é necessário adicionar o atributo href="" e colocar a url entre as aspas:

<a href="https://github.com/marcella2808">Meu GitHub<a>

Para abrir o link em outra aba, podemos usar o atributo target="" com o valor _blank:

<a href="https://github.com/marcella2808" target="_blank">Meu GitHub<a>
